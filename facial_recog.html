Eigenfaces for face recognition.¶

Implementing the Eigenface method for recognizing human faces using face images from The Yale Face Database B. Dataset includes 64 images under different lighting conditions for each of 10 distinct subjects.

In [5]:
rank_approx = []
rank = range(1,201)
for r in rank:
    x_r = np.dot(np.dot(u[:,:r],np.diag(s)[:r,:r]), vt[:r,:])
    rank_approx.append(np.linalg.norm(train_data - x_r))

plt.title('Rank-r Approximation (r: 1 to 200)')
plt.xlabel('r')
plt.ylabel('Approx. Error')

plt.show()

Calculating the feature vector:¶

In [6]:
test_u,test_s,test_vt = np.linalg.svd(test_data)

def feature_vector(data, vt, r):
    v = np.transpose(vt[:r,:])
    return np.dot(data,v)
